## 🚀 Projects Overview

### 1. Casas & Propriedades (Final Project)
A complete Real Estate management platform built during my Full Stack bootcamp 
at Codemaster.

**Core Tech:** HTML, CSS, PHP, MySQL, JavaScript, Bootstrap

**Key Features:**
- **Dynamic Listings:** All property listings and images are managed via a database and rendered dynamically
- **Responsive Carousels:** Custom image carousels per property, with separate layouts for desktop and mobile
- **Custom Backoffice:** Administrative area protected by login, with CRUD operations for properties, companies, banners and pages. Some modules (highlights, projects) partially implemented due to incomplete briefing specifications.
- **File Manager:** Upload and manage property images directly from the backoffice, with file paths stored as JSON arrays in the database
- **Database Integration:** PHP-driven logic with MySQL to fetch and render all content dynamically

### 2. MediTrust
A project focused on mastering Backend logic, security, and asynchronous operations
by using a pre-built UI template.

**Core Tech:** HTML, CSS, PHP, MySQL, JavaScript (Fetch API), PHPMailer, Google reCAPTCHA v2

**Key Features:**
- **Asynchronous Forms:** JavaScript Fetch API handles form submissions without page reloads
- **Data Integrity:** Robust PHP validation and secure SQL storage for medical appointments and contact forms
- **Email Confirmation:** Automated HTML emails sent via PHPMailer + Gmail SMTP upon form submission
- **Bot Protection:** Google reCAPTCHA v2 integrated on all public forms
- **Security:** Sensitive credentials protected with dotenv, PDO prepared statements against SQL Injection, server-side whitelist validation
