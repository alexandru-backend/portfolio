## 🚀 Projects Overview

### 1. Casas & Propriedades (Final Bootcamp Project)
A complete Real Estate management platform built from the ground up based on 
professional requirements.

**Core Tech:** PHP, MySQL, JavaScript, Bootstrap

**Key Features:**
- **Dynamic Content:** All company info, banners, and property listings are managed via a database
- **Custom Backoffice:** A full administrative area to Create, Read, Update, and Delete (CRUD) companies, properties, and carousel images
- **Database Integration:** PHP-driven logic to fetch and render data dynamically on the frontend

### 2. MediTrust
A project focused on mastering Backend logic, security, and asynchronous operations 
by using a pre-built UI template.

**Core Tech:** PHP, MySQL, JavaScript (Fetch API), PHPMailer, Google reCAPTCHA v2

**Key Features:**
- **Asynchronous Forms:** JavaScript Fetch API handles form submissions without page reloads
- **Data Integrity:** Robust PHP validation and secure SQL storage for medical appointments and contact forms
- **Email Confirmation:** Automated HTML emails sent via PHPMailer + Gmail SMTP upon form submission
- **Bot Protection:** Google reCAPTCHA v2 integrated on all public forms
- **Security:** Sensitive credentials protected with dotenv, PDO prepared statements against SQL Injection, server-side whitelist validation
