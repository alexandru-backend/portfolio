# Portfolio - Backend & Full Stack Projects

This repository contains my main web development projects, showcasing my journey from Full Stack training to specialized Backend development.

## 🚀 Projects Overview

### 1. Casas & Propriedades (Final Bootcamp Project)
A complete Real Estate management platform built from the ground up based on professional requirements.
* **Core Tech:** PHP, MySQL, JavaScript, Bootstrap.
* **Key Features:** * **Dynamic Content:** All company info, banners, and property listings are managed via a database.
    * **Custom Backoffice:** A full administrative area to Create, Read, Update, and Delete (CRUD) companies, properties, and carousel images.
    * **Database Integration:** PHP-driven logic to fetch and render data dynamically on the frontend.

### 2. MediTrust (In Progress)
A project focused on mastering Backend logic and asynchronous operations by using a pre-built UI template.
* **Core Tech:** PHP, MySQL, JavaScript (Fetch API).
* **Objective:** To bypass frontend design and focus 100% on server-side architecture.
* **Current Focus:**
    * **Asynchronous Forms:** Implementing JavaScript Fetch API to handle form submissions without page reloads.
    * **Data Integrity:** Robust PHP validation and secure SQL storage for medical appointments and contact forms.
